# LocalCast

## Запуск сервера
1) pip install fastapi uvicorn  
2) python server.py  
3) Если всё ок, в консоли будет:
   Uvicorn running on http://0.0.0.0:8000  
4) Проверка сервера:
   http://127.0.0.1:8000/health  
   В браузере должно быть написано: OK

## Как узнать IP компьютера
Mac:
ipconfig getifaddr en0  

Windows:
ipconfig  

(нужен IPv4 Address)

## Подготовка видеоконтента
1) В папке проекта создайте папку `tv_content`
2) Внутри неё:
   - `common/` — видео для всех телевизоров
   - `tv_1/` — видео только для ТВ 1
   - `tv_2/` — видео только для ТВ 2

Пример:
tv_content/
- common/
- tv_1/
- tv_2/

## Подключение VLC на Android TV
1) Компьютер и телевизор должны быть в одной локальной сети
2) В VLC выберите «Открыть сетевой поток»
3) Введите:
   http://IP_КОМПЬЮТЕРА:8000/playlist/1.m3u

Где `1` — ID телевизора (папка `tv_1`)

## Обновление видео
После добавления или удаления файлов:
- перезапустите воспроизведение в VLC
- либо заново откройте ссылку плейлиста
